-----------------------

1. Все проекты настроены с поддержкой OpenMP (флаг /openmp)

3. Конфигурации: Debug и Release для платформы x64
4. PlatformToolset: v143 (Visual Studio 2022, совместимо с VS 2019)
5. Все проекты - консольные приложения

Запуск:
-------

### OpenMP программы (Task02-Task12):
1. Откройте OpenMP_Labs.sln в Visual Studio 2019/2022
2. Выберите конфигурацию (Debug или Release)
3. Выберите платформу x64
4. Выберите проект для запуска (правой кнопкой -> Set as Startup Project)
5. Нажмите F5 (с отладкой) или Ctrl+F5 (без отладки)

### MPI программы (Task15-Task20):
1. Соберите проект в Visual Studio (как описано выше)
2. Запустите через bat-файл:
   cd Task15_MPI_IAm
   run.bat
   
3. Или вручную через командную строку:
   "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe" -np 3 Task15_MPI_IAm.exe


Примечания:
-----------

OpenMP:
- Для Task04_SharedPrivate требуется Windows.h (функция Sleep)
- Порядок вывода из разных нитей может меняться при разных запусках
- Для измерения производительности используйте Release-сборку
- Количество нитей можно задать через omp_set_num_threads() или num_threads()

MPI:
- Требуется установка Microsoft MPI (см. INSTALL_MPI.md)
- Программы запускаются только через mpiexec.exe
- Количество процессов задается параметром -np
- Каждый процесс имеет отдельную память
- Коммуникация только через MPI_Send/MPI_Recv

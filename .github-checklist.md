## ⚠️ Нужно выполнить ВРУЧНУЮ перед первым коммитом:

### 1. Удалить файлы, которые не должны попасть в репозиторий:

```powershell
# Запустите в корне проекта
Remove-Item -Force msmpisdk.msi -ErrorAction SilentlyContinue
Remove-Item -Force method.docx -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force x64 -ErrorAction SilentlyContinue
Get-ChildItem -Recurse -Filter "*.user" | Remove-Item -Force
```

### 2. Проверить .gitignore работает:

```powershell
git status
# Убедитесь, что в списке нет:
# - x64/ папок
# - *.user файлов  
# - msmpisdk.msi
# - method.docx
```

### 3. Создать репозиторий и сделать первый коммит:

```powershell
git init
git add .
git status  # Проверьте список файлов еще раз
git commit -m "Initial commit: OpenMP and MPI lab projects"
```

### 4. Связать с GitHub и загрузить:

```powershell
# Замените YOUR_USERNAME и YOUR_REPO на свои
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

## 🚀 Запуск заданий

Каждое MPI задание содержит файл `run.bat` для запуска:

```bash
cd Task21_MPI_Ring
run.bat
```

Для гибридных заданий устанавливается переменная окружения `OMP_NUM_THREADS`:

```bash
cd Task31_MPI_OpenMP_IAm
run.bat  # Запускает с 3 потоками OpenMP
```

### Go (`TaskGo_Lab`)

```bash
cd /path/to/repo
go run ./TaskGo_Lab/hello
# или: cd TaskGo_Lab && go run ./matrix
```
